# Инструкция по запуску:
## 1. Клонируйте код и зайдите в папку с кодом.
git clone https://github.com/ALEXKLY910/Ruddit.git

cd Ruddit
## 2. Загрузите зависимости.
npm install
## 3. Запустите.
npm run dev
## 4. Зайдите на сайт по ссылке:
http://localhost:5173
